# CalendarWidget

A customizable and minimalist Android home screen widget that displays your upcoming calendar events in a clean list format.

## Features

- **Transparent Design:** Blurs into your home screen background perfectly.
- **Customizable UI:** Adjust date and event text colors using Hex codes.
- **Adjustable Font Sizes:** Scale the text from 6sp to 30sp to suit your needs.
- **Flexible Event List:** Choose how many upcoming events you want to see (up to 10).
- **Auto-Save Settings:** All changes in the settings app are applied to the widget instantly.
- **Quick Access:** Click anywhere on the widget to open your preferred calendar application.
- **Configurable Updates:** Choose update intervals from 1 hour to once a day (at midnight).
- **Readability:** Subtle text shadowing ensures visibility on any wallpaper.

## Installation

1. Download the latest `app-debug.apk` from the releases or build it from source.
2. Install the APK on your Android device.
3. Open the app to grant calendar permissions and customize your settings.
4. Add the "CalendarWidget" to your home screen via the standard Android widget menu.

## Permissions

- **READ_CALENDAR:** Required to fetch and display your calendar events.

## Development

This project is built using:
- **Kotlin**
- **Jetpack Compose** (for the settings UI)
- **RemoteViews** (for the widget implementation)
